## Purpose

EnteleSCAN is designed for:

- Blockchain explorer views
- Validator monitoring
- Network diagnostics
- AI-powered anomaly detection
- EnteleLINK bridge visibility
- ChronoSeal proof records
- GraphVAULT event context
- Q-Presence identity events
- Governance and protocol observability
